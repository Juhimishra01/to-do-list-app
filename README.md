# ✨ To-Do List Web App

A modern and responsive To-Do List web application that helps users manage daily tasks efficiently with a clean and interactive interface.

---

## 🚀 Features

- Add new tasks with priority and due date  
- Edit existing tasks  
- Delete tasks  
- Mark tasks as completed  
- Search tasks in real-time  
- Filter tasks (All / Pending / Completed)  
- Data stored using Local Storage  
- Notification system for actions  
- Smooth UI with animations and hover effects  

---

## 🖥️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Local Storage API  

---

🧠 How It Works
Tasks are stored in the browser using Local Storage
Each task includes:
Task text
Priority (Low, Medium, High)
Due date
Completion status
The UI updates dynamically using JavaScript
